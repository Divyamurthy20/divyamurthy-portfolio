#  AI/ML Engineer Portfolio – Divya Sree Murthy

A modern, responsive portfolio website showcasing my work as an AI/ML Engineer, including projects in computer vision, NLP, RAG systems, and generative AI.

---

##  Live Demo

👉 https://divyamurthy-portfolio.vercel.app/

---

##  Features

*  Modern UI with glassmorphism design
*  Fully responsive (mobile + desktop)
*  Smooth scroll animations (Intersection Observer)
*  Project showcase with real-world AI systems
*  Focus on production-level ML work
*  Contact section with email integration
*  Social links (GitHub, LinkedIn)

---

##  Tech Stack

* HTML5
* CSS3 (Custom styling, gradients, glass UI)
* JavaScript (Vanilla JS)
* Intersection Observer API

##  How to Run Locally

1. Clone the repository:

```
git clone https://github.com/Divyamurthy20/divyamurthy-portfolio.git
```

2. Open the folder

3. Open `index.html` in browser(right click on index.html and select browser)
   OR use Live Server

---

##  Deployment

This project is deployed using Vercel.

Steps:

1. Push code to GitHub
2. Import repo in Vercel
3. Deploy instantly
